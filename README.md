## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. GEMINI_API_KEY=YOURAPIKEY
4. Run the app:
   `npm run dev`
   press o + enter
